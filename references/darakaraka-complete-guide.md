## 一、Darakaraka 是什么？

### 定义
在 Jaimini 占星系统中，**Darakaraka（DK）** 是 Chara Karaka 中度数最低的行星（7星系统）或第7高的行星（8星系统，第8颗被排除），代表**配偶征象星**。

### DK 的核心意义
1. **配偶特质**：DK 的行星身份揭示了伴侣的核心性格特质
2. **灵魂层面的寻找**：DK 代表灵魂在伴侣中寻找的特定品质，超越性别，反映个人的独特需求和欲望
3. **前世因缘**：DK 的位置揭示与灵魂伴侣之间的深层因缘——前世的联系、未解决的问题、此生需要共同学习的课题
4. **吸引力维度**：DK 的特质不仅限于伴侣，也代表你被什么样的人吸引（包括非恋爱关系中的欣赏）

### 性别补充（Parashara 系统叠加）
- 女性：木星 = 丈夫征象星（需结合昼夜区分法）
- 男性：金星 = 妻子征象星
- DK **超越性别**：不论男女，DK 都代表灵魂层面的伴侣征象

## 三、DK 解读框架

解读 DK 需要分析以下维度：

1. **行星身份**：哪颗行星是 DK → 基本配偶原型
2. **星座**：DK 落入的星座 → 特质的具体表现方式
3. **Nakshatra**：DK 所在的星宿 → 更精细的能量表达
4. **宫位**：DK 落入的宫位 → 关系的生活领域
5. **合相/相位**：收到的行星影响 → 特质的增强或挑战
6. **分盘**：D9 Navamsa 中的 DK → 灵魂层面的伴侣形象
7. **逆行**：如 DK 逆行 → 关系中的挑战和延迟因素
8. **燃烧**：如 DK 被太阳燃烧 → 关系质量的削弱

### DK = Mars@Cancer 1°19' 解读（7星 & JH 8K 一致）

**基本配置**：
- Mars在Cancer = **落陷（Debilitated）**——行动力被情感驱动
- 12宫(Bhava)/1宫(Rasi) = 力量部分回升（Kendra位置）但也有隐秘/退隐特质
- 度数1°19' = 接近Cancer起始点，能量不稳定
- 木星（Virgo 13.82°）对Cancer有9相位 → 木星相位增强正面特质
- D9 Navamsa: Cancer = **Vargottama区间**（D1和D9同在Cancer）

**配偶画像（7星/JH 8K视角，主流解读）**：
- **情感驱动型行动者**：行动出于情感需要而非理性计算
- **极强的保护欲**：但表达方式间接、含蓄
- **可能职业**：医疗、护理、房地产、烹饪、心理咨询、家居设计
- **落陷的化解**：1宫Kendra+木星9相位 = 一定的力量恢复（但不如入庙强）
- **Neecha Bhanga检查**：Mars@Cancer落陷，需检查是否有取消条件

**婚姻窗口影响（7星/JH 8K系统）**：
- **2026.06-10**：木星入庙Cancer = **直临DK Mars** = 极强激活
  - 这是最强烈的婚姻触发窗口
  - Mars落陷+12宫 = 可能是内在情感波动/隐秘桃花/灵性层面的联结
- **2028.07+**：Scorpio Chara Dasha = Mars统治星座 = DK直接掌权

### Sanjay Rath 8星 DK = Sun@Aries 3°31'（备选参考）

**基本配置**：
- Sun在Aries = **入庙（Exalted）**——极强权威和领导力
- 9宫 = 高等知识/哲学/长途旅行/法律
- D9 Taurus = 金牛座（非入庙）
- Ashwini Nakshatra = 开创者、先行者、治愈者

**配偶画像（S.Rath视角，次要参考）**：

- **入庙极强** = 配偶征象品质极高
